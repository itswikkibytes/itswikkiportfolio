# Airbnb Clone

A responsive Airbnb-inspired web application built using modern web technologies. This project recreates the Airbnb user interface with a clean design, responsive layouts, reusable components, and an intuitive user experience.

## Features
- Responsive Design
- Modern UI/UX
- Property Listing Cards
- Navigation Menu
- Mobile Friendly
- Clean Code Structure
- Firebase Hosting Support

## Technologies Used
- HTML5
- CSS3
- JavaScript
- React.js
- Firebase
- Git & GitHub
